Toast

Description

Toast is a simple recipe that is enlessly customizable. If you have a toaster and bread, you can make it. If this is your first time, I recommend a medium amount of browning, similar to what is seen in the above image. But everyone has there own preferences, so feel free to experiment!

Ingredients

  • Bread
  • Butter (Optional)

Directions

  1. Slice the bread, if it isn't already
  2. Butter the bread (optional)
  3. Place in toaster
  4. Select setting
  5. Start the toaster
